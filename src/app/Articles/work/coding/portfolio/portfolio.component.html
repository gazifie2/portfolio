The place I can show you my skills

This website is a personal project, made as a place to practice some ideas, showcase my work, and serve as my portfolio.

The objective is to use as little to no libraries and ready-to-use components as possible and to design every element, styling, and animation from scratch.
For hosting, I opted for free solutions, so I had to choose a free hosting service, with the shortest domain name to be not noticeable, and I had to avoid using a backend, for treating contact forms for example, since free hosting services typically only allow static websites.
